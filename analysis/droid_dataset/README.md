# DROID Dataset Distribution Analysis

Compares robolab benchmark tasks against the DROID dataset to quantify overlap in object vocabulary, per-task coverage, and task complexity distribution.

## Prerequisites

- `robolab/tasks/_metadata/task_metadata.json` — generated by `compute_task_statistics.py`
- `analysis/droid_dataset/droid_language_annotations.json` — DROID language annotations (optional, enables word-level analyses)
    - This file can be downloaded from https://huggingface.co/KarlP/droid/blob/main/droid_language_annotations.json

## Generating figures

From the project root:

```bash
python analysis/droid_dataset/droid_dataset_distribution.py
```

Figures and CSVs are saved to `analysis/droid_dataset/output/` by default.

To use a different output directory:

```bash
python analysis/droid_dataset/droid_dataset_distribution.py --save-dir path/to/output
```

To print analysis only (no figures):

```bash
python analysis/droid_dataset/droid_dataset_distribution.py --no-plot
```

## Analyses & output files

| # | Analysis | Output |
|---|----------|--------|
| 1 | Object vocabulary overlap (Jaccard / set intersection) | `object_overlap.png` |
| 2 | Per-task in-distribution score | `per_task_scores.png` |
| 3 | Complexity distribution comparison | `complexity_comparison.png` |
| 4 | Instruction word overlap (requires DROID data) | `word_overlap.png` |
| 5 | Per-task DROID word similarity (requires DROID data) | `word_similarity.png` |
| 6 | Word-level object vocabulary venn (requires DROID data) | `object_word_venn_top{50,90,100}.png` |
| 7 | Word-level object vocabulary grid (requires DROID data) | `object_word_grid_top{50,90,100}.png` |

The `topNN` suffix indicates the DROID mention coverage threshold: e.g. `top50` keeps the smallest set of DROID object words that cover 50% of all mentions, `top100` uses the full vocabulary.

Analyses 4–7 require either `droid_language_annotations.json` (auto-detected in `analysis/droid_dataset/`) or a `tasks.jsonl` file passed via `--droid-tasks`.
